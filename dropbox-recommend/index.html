推荐一款网络存储工具：Dropbox

2009-03-27
推荐

文章目录

最近LD写毕业论文焦头烂额，我帮着翻译也跟着焦头烂额，经常是她在一台电脑上写了改，改了写，我就在另外一台电脑上跟着翻了改，改了翻。这时候，同步最新文档就相当的繁琐，因为我不愿意在电脑上打开共享目录，也不想为了这个去新增一个FTP服务，于是，我找到了一款比较符合我要求的网络工具：Dropbox

Dropbox是一家网络存储网站，对免费用户提供2G的存储空间，这和其他大多数存储网站没有什么区别。只要把文件放到Public目录下，就不需验证，直接可以通过Internet下载了。不过它另外还提供了客户端软件，安装完这个客户端并和你的账号关联之后，你电脑中的Dropbox的数据文件夹就会和网站实时同步。通过这个工具，LD直接把文件放到Dropbox的文件夹下修改，修改的时候因为Word占用文档，所以网站不会同步，一旦修改完成，就会同步到网站上，而我的电脑上也装了这个客户端，关联的是同一个帐户，所以几乎没有延迟的就从网络又同步到我的电脑中。这样就不用拿着U盘跑来跑去的修改了。

现在有人把Dropbox的客户端给改进了，作成不需要安装的携带版，更加方便。下载地址在这里，作者更新速度非常快，才装了6.490版本，就看到6.491出来了，心说再等等，还没来得及装，现在已经出到6.493了。

需要注意的是，Dropbox貌似需要Microsoft .NET Framework 3.5的支持，否则会报错关闭。Microsoft .NET Framework 3.5可以在这里下载到完整版，而不是微软默认的需要继续上网下载的安装器。
